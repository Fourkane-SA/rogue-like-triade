◆ patternEtage

int Etage::patternEtage[10][10]

Un tableau 2D d'entiers représentant l'étage.

Ce tableau représente les salles de l'étage. Si la valeur est de 0, il n'y a pas de salle. Si la valeur est de 1, il y a la salle de départ. Si la valeur est de 2, il y a une salle normale. Si la valeur est de 3, il y a une salle objet. Si la valeur est de 4, il y a la salle de boss.
